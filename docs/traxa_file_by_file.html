1. main.dart

lib/main.dart
الوظيفة: نقطة دخول التطبيق. ينشّط الـ binding، يحمل إعدادات السيرفر، يثبت SSL override، يهيئ كل الـ services عبر GetIt، ثم يشغل MyApp. كذلك فيه AuthGate اللي يحدد الشاشة الأولى حسب حالة الـ user.

الـ Functions / Widgets

main() — أول دالة تتنفذ:
  1. WidgetsFlutterBinding.ensureInitialized(): تفعيل جسر Dart↔C++ قبل أي async.
  2. ServerConfig.load(): قراءة URL السيرفر المحفوظ.
  3. HttpOverrides.global = MyHttpOverrides(): ثقة SSL بـ host واحد.
  4. initServiceLocator(): تسجيل وتهيئة كل الـ singletons.
  5. runApp(MyApp()): تشغيل الـ UI.
MyHttpOverrides — class تورّث HttpOverrides. تـ override الـ badCertificateCallback لتقبل أي شهادة لو الـ host يطابق AppConstants.baseUrl فقط. ضروري للـ self-signed certs أثناء التطوير.
MyApp (StatelessWidget) — جذر التطبيق. يلف الشجرة بـ:
  • AppProviders: يحقن AuthCubit + DataCubit + ThemeCubit.
  • BlocBuilder<ThemeCubit>: يعيد البناء عند تغير الـ theme.
  • ScreenUtilInit: designSize 393×873 (يفعّل .w/.h/.sp/.r).
  • MaterialApp: themeMode + lightTheme + darkTheme + home.
  • MediaQuery override: textScaler مقيد بين 1.0 و 1.2 لمنع كسر UI.
_AuthGate (StatelessWidget) — بواب التطبيق:
  • BlocConsumer<AuthCubit, AuthState> يستمع وstateBuilder معاً.
  • listenWhen: يفعّل listener فقط لما يكون login جديد (مش كل تغيير).
  • listener: يستدعي DataCubit.fullReload() عند login.
  • builder: لو الـ user دكتور/معيد → DoctorScreen، لو طالب → StudentScreen، غير كده LoginScreen.

  • clearToken(): مسح من الذاكرة والـ SecureStorage.
  • getToken() / hasValidToken: getters.
  • _headers getter: يرجع {'Content-Type', 'Authorization': 'Bearer X'}.

Auth

  • login(username, password): admin login.
  • studentLogin(studentId, password, isFirstLogin): login الطلاب مع flag لأول دخول (لإجبارهم على تغيير password).
  • doctorLogin(username, password): login الدكاترة.
  • logout(): يستدعي clearToken.
  • _decodeOrServerDown(response): يفحص لو الـ body مش JSON (مثلاً صفحة Cloudflare error 1033 HTML) ويرمي FormatException رسالته "Server unreachable".

  • getCurrentSemester(): يجيب 1 أو 2.

  • persistAcademicYear(year): يحفظ في SharedPrefs.

Grades / QR / Attendance

  • getStudentGrades / getStudentGradesWithToken / checkGradesStatus.
  • getGradeDistribution(doctorId, subjectId): يرجع توزيع الدرجات (midterm/oral/practical/attendance/assignment/final) أو null. الـ defaults 10/5/20/5/10/50=100.
  • getStudentQRCode / recordQRCodeScan / getDoctorQRCodes.
  • getAttendance(token).

Enrollment

  • getStudentEnrollmentStatus, getAvailableSubjectsForEnrollment, registerForSubject, dropSubject, getAcademicSettings, changeStudentPassword.

Avatar / TA Management

  • _blankPng: ثابت لـ PNG 1×1 شفاف (base64).
  • forceRemoveAvatar(kind, id, token): ترفع PNG فارغ ثم تحذف عشان السيرفر بعض الأحيان يحتاج upload قبل DELETE.
  • assignTAToSubject / removeTAFromSubject: ربط TA بمادة.
